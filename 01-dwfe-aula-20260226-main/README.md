# Meu Site de Notícias

Projeto desenvolvido durante a disciplina de **Desenvolvimento Web Front-End**, com o objetivo de praticar e aplicar conceitos de HTML5, CSS3 e JavaScript na construção de uma aplicação web completa.

## Páginas

| Página | Descrição |
|---|---|
| `login.html` | Tela de autenticação do usuário |
| `cadastro.html` | Tela de cadastro de novo usuário |
| `home.html` | Feed principal com cards de notícias |

## Tecnologias utilizadas

- **HTML5** — estrutura semântica das páginas
- **CSS3** — estilização, Flexbox e layout responsivo
- **JavaScript** — interatividade e validação de formulários
- **Font Awesome** — ícones da interface
